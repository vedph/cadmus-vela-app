## Docker

Quick Docker image build:

1. `npm run build-lib`
2. update version in `env.js` and `ng build`
3. `docker build . -t vedph2020/cadmus-vela-app:0.0.1 -t vedph2020/cadmus-vela-app:latest` (replace with the current version).
